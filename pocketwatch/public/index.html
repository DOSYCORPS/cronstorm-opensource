What is an On-Demand Pocketwatch Timer?

Flexible, fine-resolution task scheduling

On-demand Pocketwatch Timers are schedulers that can issue HTTP requests at intervals as fine as 1 second, or longer.

You can use them to:

  • schedule background tasks,
  • run jobs,
  • trigger recurrent workloads,

all the things you might use cron/crontab for, but when you want it more scalable, more fault-tolerant than one machine running cron — you use Pocketwatch Timers.
